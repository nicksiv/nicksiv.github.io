About me

Nyk0 is an experimenting cybernaut!

 

There's not much to say about myself.

I try to take things slow and make things everyday, either code, text or drawings. This website has been through lots of rebuilds using numerous of software combinations.

It is now generated just from plain HTML files, and a small and simple python script. Needless to say it is still under development.

 

Uses

I use a Thinkpad T420 w/ 8GB RAM running Debian Gnome.

I edit text in emacs and sometimes in vim.

I browse the web with Firefox-esr.

I listen to music with mpv and cmus.

I also use some other great software like cwm, tmux, nextcloud, keepassXC and newsboat

I'm also a sucker for systems like arch, openBSD and plan9 which I might dive into when I grow up!
